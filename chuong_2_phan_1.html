📝 BÀI 2.1: Hàm mật độ f(x) = c(3x - x²) (3 câu) ▼
📋 Đề bài:
Cho ĐLNN X liên tục có hàm mật độ xác suất f(x) = c(3x - x²) nếu x ∈ [0,3]; 0 nếu x ∉ [0,3].

a) Xác định hằng số c
b) Tính P(1 < X < 5)
c) Tìm hàm phân phối xác suất của X

💡 Câu a) Xác định hằng số c

Điều kiện chuẩn hóa:
$$\int_{-\infty}^{+\infty} f(x)dx = 1$$

$$\int_0^3 c(3x - x^2)dx = 1$$

$$c \left[\frac{3x^2}{2} - \frac{x^3}{3}\right]_0^3 = 1$$

$$c \left(\frac{3 \times 9}{2} - \frac{27}{3}\right) = 1$$

$$c \left(\frac{27}{2} - 9\right) = c \times \frac{9}{2} = 1$$

$$c = \frac{2}{9}$$
✅ Đáp số: $c = \frac{2}{9}$

💡 Câu b) Tính P(1 < X < 5)

Do $f(x) = 0$ khi $x > 3$, nên:
$$P(1 < X < 5) = P(1 < X \leq 3) = \int_1^3 \frac{2}{9}(3x - x^2)dx$$

$$= \frac{2}{9} \left[\frac{3x^2}{2} - \frac{x^3}{3}\right]_1^3$$

$$= \frac{2}{9} \left[\left(\frac{27}{2} - 9\right) - \left(\frac{3}{2} - \frac{1}{3}\right)\right]$$

$$= \frac{2}{9} \left[\frac{9}{2} - \frac{7}{6}\right] = \frac{2}{9} \times \frac{20}{6} = \frac{20}{27}$$
✅ Đáp số: $P(1 < X < 5) = \frac{20}{27} \approx 0.7407 = 74.07\%$

💡 Câu c) Tìm hàm phân phối xác suất của X

Hàm phân phối: $F(x) = P(X \leq x) = \int_{-\infty}^x f(t)dt$
Với $x < 0$: $F(x) = 0$

Với $0 \leq x \leq 3$:
$$F(x) = \int_0^x \frac{2}{9}(3t - t^2)dt = \frac{2}{9}\left[\frac{3t^2}{2} - \frac{t^3}{3}\right]_0^x$$
$$F(x) = \frac{2}{9}\left(\frac{3x^2}{2} - \frac{x^3}{3}\right) = \frac{x^2}{3} - \frac{2x^3}{27}$$

Với $x > 3$: $F(x) = 1$
✅ Đáp số: $$F(x) = \begin{cases} 0 & \text{nếu } x < 0 \\ \frac{x^2}{3} - \frac{2x^3}{27} & \text{nếu } 0 \leq x \leq 3 \\ 1 & \text{nếu } x > 3 \end{cases}$$
📝 BÀI 2.2: Hàm phân phối F(x) = 1/2 + (1/π) arctan x (2 câu) ▼
📋 Đề bài:
Cho ĐLNN X có hàm phân phối xác suất: $F(x) = \frac{1}{2} + \frac{1}{\pi} \arctan x$

a) Tính P(0 < X < 1)
b) Tìm hàm mật độ xác suất của X

💡 Câu a) Tính P(0 < X < 1)

$$P(0 < X < 1) = F(1) - F(0)$$

$$F(1) = \frac{1}{2} + \frac{1}{\pi} \arctan(1) = \frac{1}{2} + \frac{1}{\pi} \times \frac{\pi}{4} = \frac{1}{2} + \frac{1}{4} = \frac{3}{4}$$

$$F(0) = \frac{1}{2} + \frac{1}{\pi} \arctan(0) = \frac{1}{2} + 0 = \frac{1}{2}$$

$$P(0 < X < 1) = \frac{3}{4} - \frac{1}{2} = \frac{1}{4}$$
✅ Đáp số: $P = 0.25 = 25\%$

💡 Câu b) Tìm hàm mật độ xác suất của X

$$f(x) = F'(x) = \frac{d}{dx}\left(\frac{1}{2} + \frac{1}{\pi}\arctan x\right)$$

$$f(x) = \frac{1}{\pi} \times \frac{1}{1+x^2} = \frac{1}{\pi(1+x^2)}$$
✅ Đáp số: $f(x) = \frac{1}{\pi(1+x^2)}$ với $x \in \mathbb{R}$ (Phân phối Cauchy)
📝 BÀI 2.3: Ba xạ thủ bắn bia (2 câu) ▼
📋 Đề bài:
Ba xạ thủ độc lập bắn vào bia, mỗi xạ thủ có một viên đạn. Xác suất để từng xạ thủ bắn trúng bia lần lượt là 0,8; 0,7; 0,6. Gọi X là số viên đạn bắn trúng bia.

a) Lập bảng phân phối xác suất và tính các đặc trưng số của X
b) Tính P(2 ≤ X ≤ 7)

💡 Câu a) Lập bảng PPXS và tính đặc trưng số

X có thể nhận giá trị: 0, 1, 2, 3
Tính P(X = k):

$$P(X = 0) = 0.2 \times 0.3 \times 0.4 = 0.024$$

$$P(X = 1) = 0.8 \times 0.3 \times 0.4 + 0.2 \times 0.7 \times 0.4 + 0.2 \times 0.3 \times 0.6$$ $$= 0.096 + 0.056 + 0.036 = 0.188$$

$$P(X = 2) = 0.8 \times 0.7 \times 0.4 + 0.8 \times 0.3 \times 0.6 + 0.2 \times 0.7 \times 0.6$$ $$= 0.224 + 0.144 + 0.084 = 0.452$$

$$P(X = 3) = 0.8 \times 0.7 \times 0.6 = 0.336$$
Bảng phân phối xác suất:
X0123
P0.0240.1880.4520.336
Kỳ vọng:
$$E(X) = 0 \times 0.024 + 1 \times 0.188 + 2 \times 0.452 + 3 \times 0.336$$ $$= 0 + 0.188 + 0.904 + 1.008 = 2.1$$

Kỳ vọng X²:
$$E(X^2) = 0^2 \times 0.024 + 1^2 \times 0.188 + 2^2 \times 0.452 + 3^2 \times 0.336$$ $$= 0 + 0.188 + 1.808 + 3.024 = 5.02$$

Phương sai:
$$Var(X) = E(X^2) - [E(X)]^2 = 5.02 - (2.1)^2 = 5.02 - 4.41 = 0.61$$

Độ lệch chuẩn:
$$\sigma(X) = \sqrt{0.61} \approx 0.781$$
✅ Đáp số: $E(X) = 2.1$, $Var(X) = 0.61$, $\sigma(X) \approx 0.781$

💡 Câu b) Tính P(2 ≤ X ≤ 7)

Do X chỉ nhận giá trị 0, 1, 2, 3 nên:
$$P(2 \leq X \leq 7) = P(X = 2) + P(X = 3) = 0.452 + 0.336 = 0.788$$
✅ Đáp số: $P = 0.788 = 78.8\%$
📝 BÀI 2.4: Lấy quả cầu từ hộp (1 câu) ▼
📋 Đề bài:
Một hộp có 4 quả cầu trắng và 3 quả cầu đen. Lấy ngẫu nhiên lần lượt từng quả cầu cho đến khi lấy được quả cầu trắng. Tìm hàm phân phối xác suất cho số quả cầu được lấy ra?

💡 Giải

Gọi X là số quả cầu được lấy ra. X có thể nhận giá trị: 1, 2, 3, 4
$$P(X = 1) = \frac{4}{7}$$ (Lấy ngay quả trắng)

$$P(X = 2) = \frac{3}{7} \times \frac{4}{6} = \frac{12}{42} = \frac{2}{7}$$ (Đen, rồi Trắng)

$$P(X = 3) = \frac{3}{7} \times \frac{2}{6} \times \frac{4}{5} = \frac{24}{210} = \frac{4}{35}$$ (Đen, Đen, Trắng)

$$P(X = 4) = \frac{3}{7} \times \frac{2}{6} \times \frac{1}{5} \times \frac{4}{4} = \frac{24}{840} = \frac{1}{35}$$ (Đen, Đen, Đen, Trắng)
Hàm phân phối xác suất:
$$F(x) = P(X \leq x) = \begin{cases} 0 & \text{nếu } x < 1 \\ \frac{4}{7} & \text{nếu } 1 \leq x < 2 \\ \frac{4}{7} + \frac{2}{7} = \frac{6}{7} & \text{nếu } 2 \leq x < 3 \\ \frac{6}{7} + \frac{4}{35} = \frac{34}{35} & \text{nếu } 3 \leq x < 4 \\ 1 & \text{nếu } x \geq 4 \end{cases}$$
✅ Đáp số: Đã tìm được hàm phân phối F(x) như trên
📝 BÀI 2.5: Bảo hiểm xe máy (1 câu) ▼
📋 Đề bài:
Tỷ lệ xe máy bị tai nạn trong năm là 0,0055. Công ty bảo hiểm bán với giá 60.000đ/xe, bồi thường 3 triệu/vụ. Chi phí quản lý 30% giá bán. Tính lợi nhuận kỳ vọng cho mỗi hợp đồng?

💡 Giải

Phân tích:
  • Thu từ bán BH: 60.000đ
  • Chi phí quản lý: 60.000 × 0.3 = 18.000đ
  • Xác suất tai nạn: p = 0.0055
  • Bồi thường nếu tai nạn: 3.000.000đ
Gọi X là lợi nhuận (đơn vị: nghìn đồng)
- Nếu không tai nạn (p = 0.9945): $X = 60 - 18 = 42$
- Nếu có tai nạn (p = 0.0055): $X = 60 - 18 - 3000 = -2958$

Kỳ vọng lợi nhuận:
$$E(X) = 42 \times 0.9945 + (-2958) \times 0.0055$$ $$= 41.769 - 16.269 = 25.5 \text{ nghìn đồng}$$
✅ Đáp số: Lợi nhuận kỳ vọng = 25.500đ/hợp đồng
📝 BÀI 2.6: Xe buýt - Tìm giá vé (1 câu) ▼
📋 Đề bài:
Khảo sát số khách trên xe buýt. Chi phí 200 ngàn/chuyến. Tìm giá vé để lời bình quân 100 ngàn/chuyến.

Số khách2530354045
Tần suất0.150.20.30.250.1

💡 Giải

Gọi X là số khách trên một chuyến, a là giá vé (nghìn đồng)
Kỳ vọng số khách:
$$E(X) = 25 \times 0.15 + 30 \times 0.2 + 35 \times 0.3 + 40 \times 0.25 + 45 \times 0.1$$ $$= 3.75 + 6 + 10.5 + 10 + 4.5 = 34.75 \text{ khách}$$

Thu nhập trung bình: $34.75 \times a$ nghìn đồng

Lợi nhuận trung bình = Thu nhập - Chi phí:
$$34.75a - 200 = 100$$ $$34.75a = 300$$ $$a = \frac{300}{34.75} \approx 8.633 \text{ nghìn đồng}$$
✅ Đáp số: Giá vé cần là khoảng 8.600đ - 8.700đ
📝 BÀI 2.7: Lấy sản phẩm từ 2 hộp (1 câu) ▼
📋 Đề bài:
Hộp I: 10 SP (7 tốt, 3 phế). Hộp II: 15 SP (11 tốt, 4 phế). Lấy 1 từ I, 2 từ II. Lập bảng phân phối xác suất số SP tốt.

💡 Giải

Gọi X là số SP tốt. X nhận giá trị: 0, 1, 2, 3
Tính P(X = k):

$$P(X = 0) = \frac{3}{10} \times \frac{C_4^2}{C_{15}^2} = \frac{3}{10} \times \frac{6}{105} = \frac{18}{1050} = \frac{3}{175}$$

$$P(X = 1) = \frac{3}{10} \times \frac{C_{11}^1 \times C_4^1}{C_{15}^2} + \frac{7}{10} \times \frac{C_4^2}{C_{15}^2}$$ $$= \frac{3}{10} \times \frac{44}{105} + \frac{7}{10} \times \frac{6}{105} = \frac{132 + 42}{1050} = \frac{174}{1050} = \frac{29}{175}$$

$$P(X = 2) = \frac{3}{10} \times \frac{C_{11}^2}{C_{15}^2} + \frac{7}{10} \times \frac{C_{11}^1 \times C_4^1}{C_{15}^2}$$ $$= \frac{3}{10} \times \frac{55}{105} + \frac{7}{10} \times \frac{44}{105} = \frac{165 + 308}{1050} = \frac{473}{1050}$$

$$P(X = 3) = \frac{7}{10} \times \frac{C_{11}^2}{C_{15}^2} = \frac{7}{10} \times \frac{55}{105} = \frac{385}{1050} = \frac{11}{30}$$
Bảng phân phối:
X0123
P3/17529/175473/105011/30
Kỳ vọng:
$$E(X) = \frac{7}{10} \times 1 + \frac{11}{15} \times 2 = 0.7 + 1.467 = 2.167$$
✅ Đáp số: Kỳ vọng số SP tốt ≈ 2.17
📝 BÀI 2.8: Bán máy lạnh có bảo hành (1 câu) ▼
📋 Đề bài:
Bán máy lạnh lời 800k, bảo hành lỗ 1tr. Xác suất bảo hành 10%. Tính mức lời trung bình?

💡 Giải

Gọi X là lợi nhuận (nghìn đồng)
- Không bảo hành (p = 0.9): X = 800
- Phải bảo hành (p = 0.1): X = -1000

📝 BÀI 2.9: Lấy 3 SP từ hộp ngẫu nhiên (1 câu) ▼
📋 Đề bài:
10 hộp (4 loại I, 6 loại II). Loại I: 15 tốt/5 phế. Loại II: 13 tốt/7 phế. Chọn 1 hộp, lấy 3 SP. Tính kỳ vọng số SP tốt?

💡 Giải

Gọi X là số SP tốt.
$$P(H_1) = \frac{4}{10} = 0.4, \quad P(H_2) = \frac{6}{10} = 0.6$$

Tính P(X = 0):
$$P(X = 0) = 0.4 \times \frac{C_5^3}{C_{20}^3} + 0.6 \times \frac{C_7^3}{C_{20}^3}$$ $$= 0.4 \times \frac{10}{1140} + 0.6 \times \frac{35}{1140} = \frac{4 + 21}{1140} = \frac{25}{1140}$$

Tính P(X = 1):
$$P(X = 1) = 0.4 \times \frac{C_{15}^1 \times C_5^2}{C_{20}^3} + 0.6 \times \frac{C_{13}^1 \times C_7^2}{C_{20}^3}$$ $$= 0.4 \times \frac{150}{1140} + 0.6 \times \frac{273}{1140} = \frac{60 + 163.8}{1140} = \frac{223.8}{1140}$$

(Tương tự cho X = 2, 3)

Kỳ vọng:
$$E(X) = 0.4 \times \frac{15}{20} \times 3 + 0.6 \times \frac{13}{20} \times 3 = 0.9 + 1.17 = 2.07$$
✅ Đáp số: $E(X) \approx 2.07$ SP tốt
